1. Запустить SQL shell (psql)
2. Выбрать необходимые настройки, либо нажать enter на пунктах 1-4
3. Ввести пароль ползователя
4. Ввести команду: "\i 'Ваш полный путь к файлу скрипта в одинарных кавычках, директории разделены: /'"
пример команды - "\i 'D:/ExchangeRateLoader/SQL-scripts/CreateDBWithTable.sql'", скрипт находится в директории проекта, в папке "SQL-scripts/CreateDBWithTable.sql"
5. Открыть файл "appsettings.json" в любом текстовом редакторе, файл находится в деректории проекта, в папке "ExchangeRateLoader", 
заменить значение свойства "DefaultConnection" на строку подключения к вашей базе данных
пример строки подключения - "Host=localhost;Port=5432;Database=exchange_rate;Username=postgres;Password=343454640"
6. Открыть решение в Visual Studio, запустить проект, ввести свойства для загрузки значений в базу
пример: дата - 21.11.2020, валюта - USD
